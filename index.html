Lucky Bot Command Help Center

This website is for command help with Lucky Bot for Discord. For more information, please contact Rave#0737.

Info Commands


*userinfo [@mention] or [user ID] Shows information about a user.

*serverinfo Shows information about the server.

*botinfo Shows information about the server.

*invite Sends a direct message to the requester with information on how to add Lucky Bot to a server.

*trello Sends a link to Lucky Bot's trello page.

*github Sends a link to Lucky Bot's github page.

*issue [message] Allows the user to report an issue they are experiencing with Lucky Bot.

*suggestion [message] Allows the user to suggest a feature they would like to see on Lucky Bot.

Profile Commands


*profile [@mention] or [user ID] Shows the profile of the user.

*ticket Adds a ticket to your account. Can be used once every 6 hours.

*rep [@mention] or [user ID] Gives the user a reputation point. Can be used once every 12 hours.

Last FM


*lastfm Shows basic account info.

*lastfm set Saves lastfm username for future use.

*lastfm nowplaying Shows the song currently playing.

*lastfm recent Shows the most recent songs played.

*lastfm toptracks [week|month|3-month|half-year|year] Shows the top tracks of [period].

*lastfm topartist [week|month|3-month|half-year|year] Shows the top artists of [period].

*lastfm topalbums [week|month|3-month|half-year|year] Shows the top albums of [period].

Notifications


*notify add [keyword] Adds a [keyword] to notify you about on the server.

*notify remove [keyword] Removes a [keyword] you were notified about on the server.

*notify list Direct messages a list of keywords for the server.

*notify clear Removes all keywords for the server.

*notify global add [keyword] Adds a [keyword] to notify you about on all servers.

*notify global remove [keyword] Removes a global [keyword].

*notify global list Direct messages a list of global keywords.

*notify global clear Removes all global keywords you have.

*notify ignore channel [#channel-name or channel ID] Ignores all keyword triggers in the specified channel.

*notify ignore server Ignores all global keyword triggers in the server.

Custom Commands


*command list Direct messages a list of custom commands on the server.

*command add [name] [command] Adds a custom command to the server.

*command remove [name] [command] Removes a custom command on the server.

*command edit [name] [command] Edits a custom command on the server.

Music


**nowplaying shows the song that is currently playing

**play [title|URL|subcommand] plays the provided song

**playlists shows the available playlists

**queue [page number] shows the current queue or specified page number of queue

**remove [position|all] removes a song from the queue

**search [query] earches Youtube for a provided query

**scsearch [query] searches Soundcloud for a provided query

**shuffle shuffles the queue

**skip initiates a vote to skip the current song

**forceskip skips the current song (DJ role required)

**pause pauses. duh. (DJ role required)

**repeat [on|off] toggles re-adding music to the queue when it is empty

**skipto [position] skips to the specified song (DJ role required)

**stop stops and clears the queue (DJ role required)

**volume [0 - 150] sets volume or displays current volume if no extra argument is given (DJ role required)

**setdj [rolename|none] sets the DJ role for certain music commands (Admin role required)

**settc [channel-name|none] sets the text channel for music commands (Admin role required)

**setvc [channel-name|none] sets the voice channel for playing music (Admin role required)

Greeter


*greeter channel [#channel-name or ID] Sets the channel the bot should greeter new members in.

*greeter [message] Sets the message the bot says when a new member joins. Adding {server} will display the server name, {user} will display the user's name and tag, and {mention} will mention the user.

Logging and Roles


*start roles [#channel-name or ID] Sets the channel for the role system.

*start logs [#channel-name or ID] Sets the channel for message logs.

*start join [#channel-name or ID] Sets the channel for users joining and leaving (message cannot be changed through this command).

Toggles


*toggle image Changes between embed and non-embed for images in message logs.

*toggle logs Turns message logs on and off.

*toggle greeter Turns greeter message on and off.

*toggle roles Turns roles channel on and off.

Misc


*color [hex code|random] Returns an image of the color provided or a random color.

Mod Commands


*setprefix [prefix] Changes the prefix for Lucky Bot.

*autorole [rolename] Sets a role to be added to a user when they join the server.

*ban [@user or ID] [days] [reason] Bans a user and removes the messages from [days] for [reason]. Days default is 0.

*kick [@user or ID] [reason] Kicks a user for [reason].

*prune [number of messages] Deletes [number of messages] as well as the command line.

Relays


Relays connect a two way chat system between two channels on a server or between two servers. Please contact Rave#0737 or OrigamiCoder#1375 to have one set up.
