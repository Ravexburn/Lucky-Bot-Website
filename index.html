Lucky Bot Command Help Center

This website is for command help with Lucky Bot for Discord. For more information, please contact Rave#0737.

Info Commands


*userinfo [@mention] or [user ID] Shows information about a user.

*serverinfo Shows information about the server.

*botinfo Shows information about the server.

*invite Sends a direct message to the requester with information on how to add Lucky Bot to a server.

*trello Sends a link to Lucky Bot's trello page.

*github Sends a link to Lucky Bot's github page.

*issue [message] Allows the user to report an issue they are experiencing with Lucky Bot.

*suggestion [message] Allows the user to suggest a feature they would like to see on Lucky Bot.

Profile Commands


*profile [@mention] or [user ID] Shows the profile of the user.

*ticket Adds a ticket to your account. Can be used once every 6 hours.

*rep [@mention] or [user ID] Gives the user a reputation point. Can be used once every 12 hours.
